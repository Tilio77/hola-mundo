### Introduccion al repositorio

*Este repositorio consiste en un reto personal, podria considerarse mi "Hola Mundo".*
*Es un formulario con un background image, un contenedor que hace uso de width y height con un efecto blur que contiene el formulario, efectos hover en el button, media queries para que sea responsive a distintos dispositivos y de mas efectos que estan en el archivo styles.css.*
